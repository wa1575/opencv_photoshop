# opencv_photoshop
the mini photoshop using qt by cpp


## 영상처리 기말고사 실습
## 환경 : Qt C ++ 5.15.2, openCV 3.3.1, MSVC 2019 64 비트



## 현재 할 수 있는 작업 
 사진 확대 및 축소
 
 흐리게, 선명하게
 
 대비, 밝기, 회색조 조정, 노멀라이즈, 경계 검출 
 
 모폴로지 연산
 
 Histogram
 
 아핀 회전, 원근 변환
